Site Stack

This site was developed using a combination of the following languages and utilities:

  • HTML & CSS
  • Tailwind CSS
  • Jekyll: Jekyll was chosen as the static site generator tool of choice, as it allows me to use the Liquid templating engine to rapidly generate pages and add to this site.
  • Github: This was chosen due to the widespread usage, as well as student benefits exceeding that of any other repository management tool.
  • Github Pages: Github Pages was used as a hosting tool for serving up the static pages generated by Jekyll and Travis CI. It provides a free hosting service, with the ability to use SSL and custom domains.
  • Travis CI: Travis CI was chosen as the build-management tool of choice for this project, as Github Pages doesn't support a series of Jekyll plugins which may be needed at a later development stage.
  • Dependabot: Dependabot was enabled on deployment to ensure that all dependencies (Within Gemfile.lock) are kept updated.
