Unit 6

Unit 6 explores the usage of Operating Systems (System Software).

Learning Objectives

  • Appreciate the functionality, features and tools of OS.
  • Identify the select an operating system that is most suitable to accomplish a task.
  • Demonstrate the data storage and data access supports OS can provide.

Reading

Read Chapter 3 of your core text book: Brookshear, J. and Brylow, D. (2018), Computer Science - An Overview. 13th Ed. Harlow: Pearson.

Reflection

During this module, we have explored the functionalities of Operating Systems, as well as key factors which are common across different variants (Windows, Linux, OSX). Initial development tasks within the Python language have been explored in preparation for the first assignments.
